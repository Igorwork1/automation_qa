# Тестовое задание

## Проверка требований

1. Таблица в консоли, данные корректно — ✅
2. Заголовок таблицы — ✅
3. Заголовки столбцов — ⚠️
4. Ровные столбцы с `|` — ✅
5. Даты в одном формате — ✅
6. Формат даты `YYYY-MM-DD HH:MM:SS` — ✅
7. Работа в Linux и Windows — ❌ (на linux не смотрел)
8. Учёт ширины консоли и обрезка, если данные не помещаются — ⚠️
9. Поддержка любой кодировки файла — ⚠️
10. Поддержка файла размером до 1 GB — ❌
11. Аргумент `-i` / `--input` для пути к файлу — ✅
12. Аргумент `-i` / `--input` для URL — ✅
13. Тесты — ❌

python script.py --input файл.txt
